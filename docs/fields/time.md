# Time
A simple field for selecting the time.

## Properties

| Property |  Type  |               Description               |
|:--------:|:------:|:---------------------------------------:|
|   label  | String | The text display above the time picker. |

## Usage
```json
// forms/example.json

{
    "$schema": "../form-schema.json",
    "id": "example",
    "name": "Example Form",
    "description": "Starter Example Form",
    "items": [
        {
            "type": "time",
            "label": "Select the Time"
        }
    ]
}
```
